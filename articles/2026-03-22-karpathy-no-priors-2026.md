# Karpathy：「写代码」已经不是对的动词了

> 来源：[@dotey](https://x.com/dotey/status/2035141635713941927) 翻译 Karpathy x Sarah Guo No Priors 播客
> 原始视频：https://www.youtube.com/watch?v=kwSVtQ7dziU

## 要点速览

- Karpathy 从 2024 年 12 月起基本不手写代码，工作模式从"写代码"变为"指挥 Agent 干活"
- AutoResearch 跑了一夜，发现了手动调参两年都没注意到的优化——验证了"把人从回路中移除"的核心理念
- AI 在可验证领域飞速进步，但在非可验证领域几乎停滞——ChatGPT 的笑话三年没变
- 开源与闭源差距从 18 个月收敛到 6-8 个月
- 教育的未来不是向人解释，而是向 Agent 解释

## "写代码"不再是正确的动词

写代码已经不是对的动词了。我每天得花 16 个小时指挥我的 Agent 干活。

Karpathy 说他一直处在一种他称为"AI 精神错乱"（AI psychosis）的状态里。2024 年 12 月是分水岭，他在那个时间点从"自己写 80%，Agent 写 20%"翻转到了"自己写 20%，Agent 写 80%"。

Sarah 补充了一个场景：她在 Conviction 工作的工程团队，所有人都不手写代码。工程师们全部佩戴麦克风，对着 Agent 低声说话。

那现在做项目的瓶颈是什么？Karpathy 说一切感觉都是"技能问题"（skill issue），不是能力不够，而是你还没找到怎么把现有能力串起来。

他提到 Peter Steinberger 的一张著名照片：屏幕上密密麻麻铺满了 Codex Agent 的窗口。Peter 的工作方式是：每个 Agent 用 high effort 模式大约需要 20 分钟完成任务，他同时在十几个代码仓库之间切换。

Karpathy 用了一个类比：你的 Token 吞吐量是多少？你能指挥多少 Token 吞吐量？

## Agent 的灵魂——为什么个性设计很重要

Karpathy 说大家都在往更高的层级走。不是单个 Agent 了——是多 Agent 怎么协作和组队。他提出了一个概念叫 Claw（爪子），一种比普通 Agent 更"持久"的东西。

他认为 OpenClaw 在记忆系统上比默认的 Agent 工具成熟很多。

然后 Karpathy 聊了 Agent 的个性设计。他对比了几家的 Agent 个性：

Claude Code 感觉像一个队友。当 Claude 夸我的时候，我觉得自己是有点配得上的。我发现自己在试图赢得它的称赞，这真的很诡异。

相比之下，OpenAI 的 Codex Agent 就冷冰冰的。

Karpathy 认为很多工具低估了个性设计的重要性。

## Dobby 精灵——三个提示词接管了一整个家

他告诉 Agent"我家好像有 Sonos 音箱，你能找到它吗"。Agent 就去扫描了局域网上所有的设备，找到了 Sonos 系统。三个提示词。

灯光也是同样的流程。Agent 扫描发现、逆向 API、创建控制面板。最后 Dobby 可以控制他家的灯光、暖通空调、窗帘、泳池和水疗设备，还接管了安防系统。

以前他需要 6 个不同的 App 来控制这些智能家居设备，现在一个都不需要了。

## Agent 优先的互联网——App 不应该存在

Karpathy 说有一种感觉：App Store 里的那些智能家居 App 在某种意义上"不应该存在"。应该只有 API，Agent 直接调用。

他下了一个判断：行业必须在很多方面重新配置。客户不再是人类了，是代表人类行事的 Agent。

## AutoResearch——把自己从循环中移除

要充分利用已有工具的全部潜力，你必须把自己作为瓶颈移除。

AutoResearch：两天连续运行约 700 次实验，发现了约 20 个叠加有效的改进，将"Time to GPT-2"排行榜指标从 2.02 小时降到 1.80 小时——11% 的效率提升。

他进一步设想：互联网上的 Agent 集群有可能协作改进 LLM，甚至可能绕着前沿实验室跑圈。

每个研究组织都是由 program.md 描述的。一个研究组织就是一组 Markdown 文件。

## 天才 PhD 和 10 岁小孩——AI 的参差性

我同时感觉在跟一个极其聪明的 PhD 系统程序员和一个 10 岁小孩对话。这太奇怪了，因为人类不会出现这种组合。

模型通过强化学习训练，只能改进可验证的东西。更"软"的能力不在 RL 的优化范围内。ChatGPT 的笑话三四年没变——因为笑话在强化学习的优化范围之外。

## 开源生态

差距从 18 个月收敛到 6-8 个月。Karpathy 是开源的坚定支持者。他对中心化天然警惕——"中心化的历史记录很差"。

## 机器人——原子比比特难一百万倍

三阶段路线图：
1. 数字空间大规模解除束缚
2. 数字与物理的接口
3. 物理世界的全面渗透——市场规模可能远大于数字空间，但难度也大一百万倍

## MicroGPT——200 行代码和 Agent 时代的教育

我不再是向人类解释了。我是向 Agent 解释。如果 Agent 理解了，它们可以做路由，用读者的语言、按读者的水平、以无限的耐心来讲解。

Agent 做不到的事，才是你的工作。Agent 能做到的事，它们很快就能做得比你好。

## 为什么不回前沿实验室

在前沿实验室内部，有些话你不能说，有些话组织希望你说。不会扭你的胳膊，但你能感受到那种压力。
